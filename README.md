## accessibility features

Built with accessibility in mind:

- **WCAG AAA compliance** - exceeds standard accessibility requirements
- **keyboard navigation** - full site functionality without a mouse
- **screen reader support** - proper ARIA labels and semantic structure
- **color contrast** - high contrast ratios for better readability
- **reduced motion** - respects user preferences for motion sensitivity
- **skip links** - quick navigation for assistive technology users
- **focus management** - clear visual indicators for interactive elements

**Special thanks to [Amber Fritz](https://amberfritz.com) for igniting my passion for accessibility. Her advocacy taught me how crucial inclusive design is, and this site reflects that commitment. Gracias, amiga.**
